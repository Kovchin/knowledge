В этом разделе рассмотрим селекторы их виды и области применения.

Вернуться на главную.

Селектор элемента

HTML CSS
Описание. Селектор элемента совпадает с названием тега. Применяется когда нужно во всем документе задать одинаковый вид всем тегам определенного класса.
Демонстрация

Какой то текст
Код 
<p>Какой то текст</p>
 p{
color: red;
}

Селектор класса

HTML CSS
Описание Селектор класса создается в HTML документе (называется желательно со смысловой привязкой к конечному результату изменений верстки).
Демонстрация

Какой то текст
Код 
<p class="red">Какой то текст</p>
 .red{
color: #f00;
}

Селектор ID

HTML CSS
Описание Селектор ID. Уникален в документе. Используется если требуется сделать ссылку на элемент. Используется в Java скрипте для доступа к элементу. Проверить можно (index.html#red)
Демонстрация

Какой то текст
Код 
<p id="red">Какой то текст</p>
 #red{
color: #0F0;
}

Контекстный селектор класса.

HTML CSS
Описание Контекстный селектор класса. Используется когда нужно прикрутить какую то логику к стилю форматирования к примеру в когда в теге P мы встречаем тег SPAN и при этом условии применяем форматирование указанное в CSS.
Демонстрация

Какой то текст


Какой то текст
Код <p>Какой <span>то</span> текст</p> <br>Какой <span>то</span> текст p span{
color: #00F;
}

Селектор параметров.

HTML CSS
Описание Селектор параметров используется когда нужно осуществить верстку в зависимости от параметров класса используемого тега.
Демонстрация

Имя:

Код <form> <div> <p>Имя: </p> <input type="text" name="name"> </div> <div> <input type="submit" name="name"> </div> </form> input[type="text"]{ height: 40px; width: 400px; } input[type="submit"]{ cursor: pointer; height: 50px; width: 300px; color: @0099dd; } input[type="text"]:focus{ border: 2px solid #c00; }

Селектор отношений.

HTML CSS
Описание Селектор тоношений. Используется когда нужно учитывать взаимное отношение тегов и в зависимости от этого осуществлять верстку.
Демонстрация

Параграф в заголовке 3-го уровня

Просто параграф
Код <h3><p>Параграф в заголовке 3-го уровня</p></h3> <p>Просто параграф</p> h3 > p { color: #FFFF00; }

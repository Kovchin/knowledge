Вводное слово в CSS.

CSS - язык в котором обпеределяются стили как будет отображаться HTML страницы.

Для подключения библиотеки main.css в корневом каталоге требуется в файле index.html в разделе head ввести следующий код

<link rel="stylesheet" href="main_css.css"/>

В то же время для подключения дополнительных библиотек в файле css требуется прописать

@import url("color.css");

Для подключения шрифтов требуется в main.css прописать

Пример использования подключаемого шрифта

@font-face{
font-family: "micra";
src: url("fonts/micra.woff") format("woff"), url("fonts/micra.woff2") format("woff2");
}

body{
font-family: micra;
}

Описание тегов CSS

border: 2px solid #000; - задает стиль рамки ширина в пикселях, стиль начертания рамки, цвет. Есть так же border-left/right/top/bottom. Solid- сплошной, dotted - из точек, dashed - из тире.

border-radius: 5px; - закругление границ рамки.

display - Многоцелевое свойство, которое определяет, как элемент должен быть показан в документе.

font-family: "Название шрифта"; - задается шрифт которым будет отображаться текст.

font-size: 150%; - размер шрифта в данном случае в процентов от нормы принятой для этого тега.

font-weight: bold; - толщина линий шрифта.

font-style: italic; - стиль шрифта в данном случае курсив.

text-align: justify; - выравнивание текста. (в данном случае заполнение всей страницы).

text-decoration: underline; - декорация текста (в данном случае подчеркивание) (none - без декораций).

text-indent: 20px; - задает отступ для красной строки абзаца.

text-transform: uppercase; - подменяет все символы текста в заглавные буквы.

text-shadow: 5px 4px 5px #555 - задает тень для текста.(смещение по горизонтали, смещение по вертикали, радиус размытия, цвет тени)/

color: #C00; - задается цвет текста в шестнадцатеричной системе R-XX G-YY B-ZZ.

line-height: 300% - задает высоту строки (расстояние между соседними строками).

list-style-type - задает стиль маркера списка (disc - залитый кружек) (cyrcle - пустой кружек) (square - квадрат) (none - без идентификатора)

list-style-image: url() - задает в качестве маркера списка картинку.

height: 600px; - задает высоту контейнера.

width: 500px; - задает ширину контейнера.

margin: 0; - поля текста сверху и снизу снаружи блока (в данном примере они отсутствуют) если поставить 20px auto то будет с отступами 20 сверху снизу и строго по центру.

padding: 0; - поля внутри блока (в данном случае они обнулены) можно указывать одной командой 4 разных значения 1px 2px 3px 4px )1 сверху 2 справа 3 снизу 4 слевa). 20px 30px; (в таком виде 20 сверху/снизу 30 слева/справа) Либо указать непосредственно padding-left/right/top/bottom

overflow: hidden; - определяет что делать с текстом вышедшем за пределы блока если его размеры жестко заданы. (hidden - скрывать, scroll; - проматывать, auto - скрлить когда выходит за рамки, visible - выводить весть текст не смотря на рамки.)

box-shadow: 5px 4px 5px #555 - задает тень блоку (смещение по горизонтали, смещение по вертикали, радиус размытия, цвет тени) есть еще одн необязательный параметр расстояние распостронения, он может быть добавлен в пикселях перед цветом.

Анимация/трансформация.

Спецификация по свойствам Transform

transition-duration: 1.5s; - задает время анимации.

transition-property: color; - свойство что будет изменяться согласно анимации (в данном случае цвет текста).

transform: rotateY(180deg); - в данном случае все разворачивается на 180 градусов в горизонтальной плоскости.

transition-timing-function: linear; - описывает вид анимации(используется редко)

transition-timing-function: steps(3); - описывает количество шагов анимации(используется редко) в данном случае 3.

@keyframes test{
0%{background-color: #F00;}
33%{background-color: #0F0;
66%{background-color: #00F;}
100%{background-color: #FFF;}
} - анимация в 4 шага изменяющая цвет фона в 4 шага. (вызывается animation-name: test;)

transform:<функция> [<функция>]* | none - Трансформирует элемент, в частности, позволяет его масштабировать, вращать, сдвигать, наклонять, а также комбинировать виды трансформаций.

transform-origin: - Устанавливает координаты точки, относительно которой будет происходить трансформация элемента.

background-position: -363px -121; - позволяет сместить начальную чточку отображения фона в комплексе с высотой и шириной блока можно подменять индикаторы списка.

backface-visibility: hidden; Определяет, показывать обратную сторону элемента или нет. Как правило, обратная сторона становится видна при трансформации элемента, например, при его вращении относительно горизонтальной или вертикальной оси в трёхмерном пространстве. В обычной ситуации свойство backface-visibility не оказывает влияния на отображение элемента.

perspective: 1000px; - задает перспективу объекту (становится видно как бы в 3d во время вращения)

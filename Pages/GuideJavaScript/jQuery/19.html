Создание новых элементов

Теория

  • $("<p>Новый абзац</p>.insertAfter('div')") - Добавление нового абзаца p после блока div

  • $("<p>Новый абзац</p>.insertBefore('div')") - Добавление нового абзаца p до блока div

  • $("div").after("<p>Новый абзац</p>") - Добавление нового абзаца p после блока div

  • $("div").before("<p>Новый абзац</p>") - Добавление нового абзаца p до блока div

И из предыдущих курсов

  • $('div').append('<p>Привет мир!</p>') - добавить ко всем блокам div параграф p в конец.
  • $('img').appendTo('#images') - переносит все img со страницы в блок с ID='images'в конец.
  • prepend - то же самое что и append только в начало блока
  • prependTo - то же самое что и appendTo только в начало блока
  • $('a').wrap('<p class="link"></p>') - оборачивает все теги a в параграфы с классом link

Задание

  • Сделайте форму с текстовыми полями, отвечающими за ширину, высоту будущего div и его цвет фона (формат ввода: #aabbcc).
  • Добавьте в форму кнопку «Добавить».
  • После нажатия на кнопку «Добавить» на странице должен появиться ниже всех остальных блоков (если таковые были) новый блок с заданной шириной и высотой, а также цветом.
  • Причём больше 5-ти блоков на странице быть не должно. Если вдруг пользователь просит добавить 6-й блок, то вывести ему через тег

    рядом с формой уведомление: «5 – это максимум!». Безусловно, вывод так же на jQuery.

$('#button').on('click', insertP);
var w, h, c, inp;

function insertP() {
inp = $('.example input');
w = inp[0].value;
h = inp[1].value;
c = inp[2].value;
if (checkInputValues(w, h)) {
var p = $('<p class="new"> Новый абзац! </p>')
p.css('width', w + 'px');
p.css('height', h + 'px');
p.css('background-color', c);
$(p.insertAfter('.example'));
}
}

function checkInputValues(w, h) {
if (w < 150) {
alert('ширина блока должна быть больше 150');
return false;
}
else if (h < 50) {
alert('Высота блока должна быть больше 50');
return false;
}
else if ($('.new').length > 4){
alert('Не более 5-ти блоков')
return false;
}

else return true;
}

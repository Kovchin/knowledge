Метод clone

Задание

  • Создайте абзац с произвольным текстом со свойством display в значении none.
  • Создайте 2 пустых блока <div>.
  • Через jQuery скопируйте текст из 1-го пункта в оба блока <div>.

Реализация

Hello World!

Код

var p = $('#paragraph').clone();
var div = $('#example div')

p.css('display', 'block');
p.css('color', 'darkred');

div.append(p);

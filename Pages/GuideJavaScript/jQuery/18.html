Удаление элементов

Теория

Есть 2 метода для удаления/очистки элементов в jQuery

  • $('img').remove() - Удаляет указанные в селекторе элементы
  • $('p').empty() - Очищает содержимое парных тегов при этом сам тег остается в DOM дереве

Задание

  • Выведите 3 изображения на страницу.
  • Под каждым изображением поместите ссылку «Удалить».
  • При нажатии на ссылку «Удалить» должны исчезнуть: изображение, соответствующее этой ссылке, и сама ссылка «Удалить».

Реализация

Убрать первую картинку
Убрать вторую картинку
Убрать третью картинку

Код

var a = $('#example a');

function removeEmpty(){
var target = this.className;
$('img.'+target).remove();
$('a.'+target).empty();
}

for(var i = 0 ; i<a.length; i++){
a[i].addEventListener('click',removeEmpty);
}

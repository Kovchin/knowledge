Знакомство с массивами

В данном видео уроке рассматривается создание массивов и немного свойств работы с этим типом данных

Инициализация массивов

Пустой массив

var mas = [];

Инициализация массива с данными

var mas = mas [1, 'test', true, [1,2], NaN];

Методы массива

  • push - вставить элемент/несколько в конец массива
    • array.push('value');
  • pop - удалить элемент с конца массива равносильно уменьшению длинны массива сокращением length
    • array.pop();
  • shift - удалить элемент с начала массива и сдвинуть все элементы на одну позицию
    • array.shift()
  • unshift - добавить элемент/несколько в начало массива и сдвинуть все элементы на необходимое число позиций вправо
    • array.unshift('value1', 'value2')

Работа с DOM

В данном уроке обсуждалась концепция построения и работы с DOM деревом

Навигация по DOM происходит посредством следующих методов

  • document.getElementById('Имя ID тега') - возвращает элемент по его идентификатору
  • document.getElementsByClassName('Имя класса') - возвращает массив со всеми элементами по Имени класса
  • document.getElementsByName('Имя элемента') - возвращает массив элементов по имени
  • document.getElementsByTagName('Тег') - Возвращает массив со всеми запрашиваемыми тегами
  • document.getElementsByTagNameNS - Х.З. пока не пользовался, а на даче нет нормального интернета что бы прочитать... :-(

Так же есть схожий механизм поиска по DOM только с применением классов CSS

  • document.querySelector('.класс') - возвращает первый элемент "класса" (здесь же может быть ссылка на #класс короче все многообразие выбора в CSS)
  • document.querySelectorAll('p') - возвращает массив элементов запрашиваемого класса либо определенные тег (к примеру запрос всех параграфов в документе)

Так же речь шла про возможность добавления и дальнейшего манипулирования элементами из JS

Для добавления элемента используется метод createElement класса document

  • var uzel = document.createElement('p'); - В частности этой строкой создается тег <p></p>

А теперь для его вывода необходимо создать узел в DOM

  • document.body.appendChild(uzel);
  • ParentElement.insertBefore(NewElement, OporniyElement);

Для демонстрации создадим параграф в который раз в 1 сек будет генерироваться случайное число

var timer = document.createElement('header');
document.body.appendChild(timer);

function checkTime(){
timer.innerHTML = Math.random();
}

setInterval(checkTime,1000);

Рассмотрим создание узла DOM после уже имеющегося на странице тега h1

var NEW = OLD = ParentNode = null;

OLD = document.querySelector('h1');
NEW = document.createElement('div');
NEW.innerHTML = '<h2> Hello world!!! </h2>';
ParentNode = OLD.parentNode;
ParentNode.insertBefore(NEW, OLD.nextSibling);

Функции и работа с ними

В уроке рассматривается теория использования функций

Создание простейшей функции без входящих данных и без вывода результата

function test(){
alert('Hello');
};

Создание функции с входными данными

function test(a, b){
console.log(a+b);
};

Созадние функции с входными данными и выводом результата

function test(a, b){
var c=a+b;
return c;
};

Назначение обработчику событий функции с параметрами (вызов через анонимную функцию)

function test(a, b){
console.log(a+b);
};
document.getElementById('name').onclick = function(){
test(12, 6);
}

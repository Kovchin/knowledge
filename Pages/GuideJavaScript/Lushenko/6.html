Операторы ветвления

В данном уроке речь идет про операторы ветвления

If и Switch :

  • if('условие 1')
    {Выполняется если условие 1 = true};
    else if ('условие 2')
    {Выполняется если условие 2 = true};
    else if ('условие 3')
    {Выполняется если условие 3 = true};
    else
    {Выполняется если все предыдущие условия не выполнились};
  • parseInt(x)
    x-текст что будет преобразован в число 'если среди текста есть цифры'
  • switch(x){
    case 'value1':
    {Выполняется если условие x = value1}
    break;
    case 'value2':
    {Выполняется если условие x = value2}
    break;
    case 'value3':
    {Выполняется если условие x = value3}
    break;
    default:
    {Выполняется если ни один из case не верно}
    break;
    }

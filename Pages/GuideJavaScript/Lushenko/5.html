Javascript вывод на страницу

В данном уроке рассматривались способы вывода информации на страницу при помощи Java Script

Задание

  • Вывести текст в теге P
  • Создать 3 кнопки для демонстрации эффектов InnerHTML/InnerText
    • 1-ый эффект - заменить текст в теге P
    • 2-ой эффект - дописать информацию к действующему параграфу
    • 3-ий эффект - тоже самое только без учета тегов
  • Создать связку кнопки radio Button для демонстрации InsertAdjacentHTML/InsertAdjacetText

9-ое задание в Курсах Михаила Русакова по Java Script

В этом уроке речь шла про 3 метода класса Window

  • alert
  • confirm
  • prompt

В качестве домашнего задания требуется запросить у пользователя целое число больше 0.

Если это число меньше, либо равно 0, то через alert() сообщите об ошибке и повторно запросите число. Делайте это до тех пор, пока пользователь не введёт нормальное число.

Рассчитайте сумму чисел от 1 до введённого числа, то есть 1+2+3+4+…+введённое_число

Код задания

<script>
var res = prompt("Введите целое число");
var result = 0;
while ((Number(res) !== res) && (res % 1 !== 0))
{
alert("Вы ввели " + res + "Это не целое число, будьте внимательнее.");
res = prompt("Введите целое число");
}
for (i = 0; i <= res; i++)
result += i;
document.write("Сумма чисел от 0 до введенного вами " + res + " = " + result);
</script>

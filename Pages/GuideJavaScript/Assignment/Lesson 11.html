11-ое задание в Курсах Михаила Русакова по Java Script

В уроке шла речь про многомерные массивы

Задания

  • Создайте двумерный массив 3 на 3 с любыми числами.

    • Код

    <script>
    var arr1 = new Array(1, 2, 3);
    var arr2 = new Array(4, 5, 6);
    var arr3 = new Array(7, 8, 9);
    var arr = new Array(arr1, arr2, arr3);
    for (i=0; i<arr.length;i++){
    for (j=0; j<arr[i].length; j++)
    document.write(arr[i][j]+ " ");
    document.write("<br>");}
    </script>

    Результат

  • Выведите этот массив в теге <table>, чтобы каждое число было в своей ячейке.

    • Код

    <script>
    var arr1,
    arr2,
    arr3 = new Array(),
    arr = new Array(arr1, arr2, arr3),
    k = 0;
    document.write("<table border=1>");
    for (i = 0; i < 3; i++) {
    document.write("<tr>")
    for (j = 0; j < 3; j++) {
    k++;
    arr[i][j] = k;
    document.write("<td>" + arr[i][j] + "</td>");
    }
    document.write("</tr>");
    }
    document.write("</table>");
    </script>

  • Напишите скрипт, который диагональные элементы сделает равными 1, а остальные 0. То есть элементы[0][0], [1][1], [2][2] – 1, а остальные 0. Сделайте это без цикла.

    • Код

    <script>
    var arr1 = new Array(1, 0, 0);
    var arr2 = new Array(0, 1, 0);
    var arr3 = new Array(0, 0, 1);
    var arr = new Array(arr1, arr2, arr3);
    document.write(arr);
    </script>
